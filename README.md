# GreenBite Orchestration

Repositorio encargado de la orquestación de la arquitectura de microservicios GreenBite mediante Docker Compose.

## Servicios incluidos

* Frontend React
* Backend For Frontend (BFF)
* Subscription Service
* User Service
* PostgreSQL

## Levantar el ecosistema completo

```bash
docker compose up --build
```

## Arquitectura implementada

* Microservicios
* Backend For Frontend (BFF)
* Database per Service
* Contenedores Docker
* Orquestación con Docker Compose
